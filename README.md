# Econometrics II

This repository contains coursework and practical exercises developed for the Econometrics II course.

## Contents

- Time series analysis using R
- Simulation of regression models with autocorrelated errors
- Applications using real-world economic data (e.g., IPCA inflation series)
- Implementation of econometric concepts in Quarto

## Tools

- R
- Quarto
- Packages: BETS, tsibble, zoo, xts

## Objective

The goal of this repository is to apply theoretical econometric concepts through coding, simulation, and data analysis.
